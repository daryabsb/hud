Update Order
{% comment %} This button directly updates the #pos element with a form that can update any field in the PosOrder model. You can customize which fields are shown in the form by adding hidden inputs with the field names: Or you can modify the hx-get URL to include query parameters that specify which fields to include: hx-get="{% url 'pos2:update-order-generic' active_order.number %}?fields=status,note,discount" The GenericOrderUpdateView will only process fields that are included in the form submission and are valid for the PosOrder model. This button targets the #pos element directly, which will update the entire POS interface when the form is submitted, similar to how order_status.html works. {% endcomment %}
